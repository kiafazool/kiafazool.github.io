Type here commands. e.g.: x = sqrt(3^2 + 4^2), sin(2*pi*x), 4*(x-0.5)^2 + 1, ...

What's new?

Version 0.1

Launched. Type a function with one independent variable and you will see a plot of that function in the range [0,1]. You can change the range by hovering the plot.

Version 0.2

Added some plot properties, such as line color, width and style. Changed to SVG format, which makes download the plot much simpler now. Added attribution operator (e.g. f = 0.1). The "x" in the upper-right corner lets you see and remove the variables in the workspace.

Version 0.3

Added more plot properties (axis labels, title and legend)

Version 0.4

Support for multiple series in the same plot. Drag the hand in the left of each series to another plot to merge the dragged series with existing ones.
Added legend to plots. The legend can be dragged to any position in the plot.

Version 0.5

Added command history. Use UP and DOWN arrow keys to scroll through the last typed commands.

Future updates

  • Add matrix support
  • Add matrix operators
  • Add more chart types (e.g. bar, pie)
  • 3D plotting (surface plots)
  • Add answer function (e.g. ans(1), ans(2))
  • Add object support ('.' operator)
