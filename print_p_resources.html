Resources used on OnMyPhD

I am using some external resources to build this website. Here I would like to show my appreciation for the developers out there that are willing to make code for free and help other people.

  • jQuery UI - a javascript library with User Interface controls

  • Flot - the plots I render in my website
  • MathJax - the equations I render in my website
  • D3.js - a SVG library
  • Byethost - A free fast and steady webhosting
